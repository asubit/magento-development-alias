MagAlias version 1.0.0 by asubit

## Description
  MagAlias is a shell tool for Magento version 1.x providing a set of development alias based on n98-magerun

## Usage
  ```
  $  @[alias]
  ```
  
## Available commands
  - `@help` Display this help text with all availables alias
  - `@edit` Edit this script source
  - `@m`    List all availables Magerun commands
### cache
  - `@cc`   Clean the Magento cache
  - `@cf`   Flush the Magento cache
### setup
  - `@si`   Compare all setup version
  - `@sr`   Run setup upgrade
